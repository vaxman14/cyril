Accessibility Statement

Effective: June 3, 2026

We are committed to making this website accessible to all visitors, including those with disabilities.

Our Commitment

This site aims to conform to the Web Content Accessibility Guidelines (WCAG) 2.1 Level AA. We work to ensure all content is perceivable, operable, understandable, and robust for users of all abilities.

What We've Done

We have taken the following steps to improve accessibility:

— All images include descriptive alt text
— Color contrast meets WCAG AA standards
— The site is navigable by keyboard
— Text is resizable without loss of content
— Semantic HTML elements are used throughout

Known Limitations

Some gallery images may have limited descriptions. We are working to improve this over time.

Feedback

If you encounter any accessibility barriers on this site, please contact us via the information on the main site. We aim to respond within 5 business days.
